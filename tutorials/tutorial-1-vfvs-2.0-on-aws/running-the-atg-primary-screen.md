# Running the ATG Primary Screen

Now we are ready to start the jobs using the following command:&#x20;

```
./vfvs_submit_jobs.py 1 1
```

Here we assumed that 1 is the number of workunits (please see the previous section on how you can obtain the correct number). The command will submit all workunits that were prepared previously with the `vfvs_prepare_workunits.py` command.&#x20;
