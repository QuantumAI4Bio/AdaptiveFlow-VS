# Using AdaptiveFlow

AdaptiveFlow is managed, used, and controlled by:

1. A number of commands in the `tools/` folder
2. The control (configuration) files
   1. &#x20;`tools/templates/all.ctrl` (before initializing the workflow)
   2. The config.json in the folder `workflow/` (after initializing the workflow)

The configuration files and the commands are described in the next subsections for AdaptiveFlow in general. Additional information on specific modules of AdaptiveFlow (AFLP, AFVS or AFU) might be present in the corresponding chapters.
