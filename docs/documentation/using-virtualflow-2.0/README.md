# Using VirtualFlow 2.0

VirtualFlow 2.0 is managed, used, and controlled by:

1. A number of commands in the `tools/` folder
2. The control (configuration) files
   1. &#x20;`tools/templates/all.ctrl` (before initializing the workflow)
   2. The config.json in the folder `workflow/` (after initializing the workflow)

The configuration files and the commands are described in the next subsections for VirtualFlow 2.0 in general. Additional information on specific modules of VirtualFlow (VFLP 2.0, VFVS 2.0 or VFU 2.0) might be present in the corresponding chapters.
