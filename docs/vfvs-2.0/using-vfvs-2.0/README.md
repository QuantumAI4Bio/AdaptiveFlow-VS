# Using VFVS 2.0

The usage of VFVS 2.0 is the same as the usage of VirtualFlow 2.0 in general, which is described [here](../../documentation/using-virtualflow-2.0/).&#x20;

In addition for VFVS 2.0 the input-files folder has to be prepared set up as described [here](../../documentation/using-virtualflow-2.0/preparing-the-workflow.md).&#x20;

All specific options/features which one can activate/configure in VFVS 2.0 are described in the control file `tools/templates/all.ctrl`, where the values for all of them can be specified. These options/features include the setting up of the docking scenarios to be used:

* Number of docking scenarios
* Docking programs and/or scoring functions to be used
* Number of replicas per docking scenario
